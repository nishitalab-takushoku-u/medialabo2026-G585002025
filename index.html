各授業回の学習事項

第1回 WWW と HTML

HTMLは要素で構成され、インライン要素の中にブロック要素を配置してはならない。要素には、段落を表すp要素などのブロック要素と強調を表すstrong要素などのインライン要素がある。また、br要素のように終了タグを持たない空要素や要素に付加情報を与えるclassやid、srcといった属性についても学んだ。<、 >、 "、 '、 &は特殊文字とされ、これらを直接表示するには特殊な記述が必要。<hx>のxに小さい数字を入れるほど、大きくて重要な見出しを作ることができる。また、リストを作る要素は様々あり、数字から始める、・から始めるなどがあり、タグの名前はliやolなど様々ある。要素の使い方に困ったらMDN Web Docsを参照する。

演習の成果:演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

外部スタイルシート（link要素での読み込み）、内部スタイルシート、インラインスタイルの3つの適用方法を学んだ。CSSファイルを使うことでHTML内の文字のフォントやサイズ、配置、背景色などを柔軟に変更することができる。リンクに触れた時に色が変わるのもこのCSSのおかげである。タグにclassで名前をつけて、コードをひとまとまりにしたり、逆に細かく分けることもできる。色はつければつけるだけいいというわけではないことがわかったので、適度に、見やすくなるような色の付け方を試行錯誤しながら見つけていきたい。

演習の成果:演習2-3:ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

第3回ではWebページに動的な変化を与えるプログラミング言語JavaScriptの基礎と外部ファイルとしてHTMLへ組み込む時のルールについて学んだ。Javaはコンパイル言語、JavaScriptはインタプリタ言語であり、コンパイルをするタイミングなどさまざまな違いがある。JavaScriptは記述ミスがあったとしてもプログラムが停止せず、NaNやundefinedといった値を返してそのまま処理が進んでしまうことがあるためJavaよりミスに気づきにくい。こういったミスを発見するために例外で停止やブレークポイントを使いこなす必要がある。またfor文をJavaScript流にすっきりまとめたfor-of構文や複数のデータをひとまとめにして管理するオブジェクトを学んだ。今後のプロジェクト課題に役立てていきたいと思う。

演習の成果:演習3-6: オブジェクトの参照
