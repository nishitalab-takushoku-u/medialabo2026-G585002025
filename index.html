各授業回の学習事項

第1回 WWW と HTML

HTMLは要素で作られる。ブロック要素の中にインライン要素は含めてはならない。<、 >、 "、 '、 &は特殊文字とされ、これらを直接表示するには特殊な記述が必要。<hx>のxに小さい数字を入れるほど、小さい見出しを作ることができる。また、リストを作る要素は様々あり、数字から始める、・から始めるなどがある。要素の使い方に困ったらMDN Web Docsを参照する。

演習の成果:演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

CSSファイルを使うことでHTML内の文字のフォントであったり、色を変えることができる。リンクに触れた時に色が変わるのもこのCSSのおかげである。タグにclassで名前をつけて、コードをひとまとまりにしたり、逆に細かく分けることもできる。色はつければつけるだけいいというわけではないことがわかったので、適度に、見やすくなるような色の付け方を試行錯誤しながら見つけていきたい。

演習の成果:演習2-3:ウェブブラウザ説明ページの装飾
